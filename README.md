# 2. 포크한 내 레포지터리 클론하기
이 레포지터리는 윈도우 환경에 클론해도 상관 없지만, wsl 리눅스 환경을 기준으로 설명합니다.

1. CMD 관리자 권한으로 실행
2. `wsl` : wsl 접속
3. `cd ~` : 홈 디렉터리로 이동
4. `git clone` `${포크한 내 레포지터리 주소}` : 원격 저장소를 로컬에 복제
5. `ls` : 현재 디렉토리의 목록 확인
6. `cd` `${포크한 내 레포지터리 이름}` : 내 레포지터리 폴더로 경로 이동
7. `code .` : VS Code로 해당 폴더 열기

    ![깃허브_클론_1](./깃허브%20클론_1.png)
